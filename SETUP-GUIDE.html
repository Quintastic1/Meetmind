🚀 Launch MeetMind from your iPad

Complete beginner guide — no coding, no computer needed. Takes about 30 minutes.

Total cost to go live: $12

Just the domain name. Everything else is free.

1

Create a free GitHub account

GitHub is where we store the app's code. Think of it like Google Drive but for websites.

→ Go to: github.com and tap "Sign up"

Use any email. Choose a username. Verify your email. That's it.

2

Create a new repository

A "repository" is just a folder for your app. Once logged into GitHub:

1. Tap the green "New" button
2. Name it: meetmind
3. Make sure it's set to Public
4. Check ✅ "Add a README file"
5. Tap "Create repository"

3

Upload the app file

You should have received a file called index.html from Claude. Now:

1. Inside your new repository, tap "Add file" → "Upload files"
2. Tap "choose your files" and select index.html
3. Scroll down and tap "Commit changes"

✓ Your app is now saved on GitHub. Next we make it live on the internet.
4

Deploy to Vercel (makes it live)

Vercel takes your GitHub code and puts it on the internet — for free.

→ Go to: vercel.com

1. Tap "Sign Up" → choose "Continue with GitHub"
2. Tap "Add New Project"
3. Find your meetmind repository and tap "Import"
4. Don't change anything — just tap "Deploy"
5. Wait ~60 seconds

✓ Vercel gives you a free link like meetmind.vercel.app — your app is LIVE!
5

Buy your domain (optional but worth it)

Instead of meetmind.vercel.app you'll want meetmind.app or similar. This costs ~$12/year.

→ Go to: namecheap.com

1. Search for meetmind.app (or meetmind.io)
2. Add to cart and checkout (~$12)
3. Back in Vercel: go to your project → Settings → Domains
4. Type your domain name and tap Add
5. Follow Vercel's instructions to connect it (copy/paste two values into Namecheap)

⏱ Domain connections take up to 24 hours to go live — totally normal.
6

Set up Stripe (to collect payments)

→ Go to: stripe.com

1. Sign up for a free account
2. Complete identity verification (takes 5 min, needs your ID)
3. You'll get a Publishable Key — save it, Claude will need it for the next version of the app

✓ Stripe is now ready to accept credit cards. You won't pay anything until you make money.
7

Set up Supabase (your database)

This stores user accounts and meeting data.

→ Go to: supabase.com

1. Sign up free with GitHub
2. Tap "New Project"
3. Name it meetmind, choose any region
4. Save the Project URL and anon key — Claude will use these next

8

You're live! Tell Claude what's next

Once you've done steps 1–7, come back to Claude and say:

"My Vercel URL is _____, my Stripe key is _____, my Supabase URL is _____, my Supabase key is _____. Now connect everything and add real payments."

Claude will then build the fully connected version with real user accounts, real payments, and real AI transcription.

✓ Your $200 budget is still mostly intact. The hard part is done.
